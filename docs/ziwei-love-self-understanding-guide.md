# 자미두수 명반 해석 가이드

> 기준: `/Users/white/Desktop/Obit/data` 디렉터리의 자미두수 자료만 사용
> 목적: 운세/조언이 아니라, 연애를 통해 나를 이해하는 자기이해형 콘텐츠를 만들기 위한 해석 기준

---

## 1. 가이드의 목적

이 가이드는 자미두수 명반을 보고 아래 4가지를 읽기 위한 기준이다.

1. 나는 관계에서 어떻게 작동하는가
2. 나는 어떤 매력으로 사람을 끌어당기는가
3. 그 매력이 과하게 발현되면 어떤 문제가 생기는가
4. 나는 어떤 관계 환경에서 자연스럽게 빛나는가

즉, 이 가이드는
**“누굴 만나야 하나”** 보다
**“나는 왜 이런 관계 패턴을 반복하나”**를 읽기 위한 해석 기준이다.

---

## 2. 해석의 기본 원칙

| 원칙 | 내용 |
|---|---|
| 자기이해 중심 | 길흉판단보다 감정 작동 방식, 관계 패턴, 매력 구조를 읽는다 |
| 단정 금지 | 살성 1~2개, 화기 1개만으로 나쁘다고 단정하지 않는다 |
| 종합 해석 | 주성, 궁위, 삼방사정, 화기, 살성, 도화, 유년/소한을 함께 본다 |
| 번역 중심 | 자미두수 용어를 그대로 던지지 말고 성향·행동·상황 언어로 번역한다 |
| 장단점 분리 금지 | 장점과 단점을 따로 보지 말고, 같은 성향의 건강한 발현/과발현으로 본다 |
| 운세 최소화 | 유년/소한은 “시기 예언”보다 “감정이 활성화되는 흐름” 정도로만 쓴다 |

---

## 3. 해석 순서

이 순서는 data 디렉터리 자료의 해석 요소들을 실무용으로 재구성한 순서다.

### Step 1. 명궁
- 타고난 기질
- 첫 반응 패턴
- 사람이 나를 처음 읽는 방식

### Step 2. 부처궁
- 연애관
- 감정을 대하는 태도
- 관계에서 기대하는 방식
- 내가 끌리는 감정의 결

### Step 3. 부처궁의 삼방사정
- **복덕궁**: 내면, 무의식, 정서 만족 기준
- **천이궁**: 외부 환경, 사회적 접점, 인연 유입 환경
- **관록궁**: 관계에 투영하는 조건, 대궁에서 오는 압력

### Step 4. 도화
- 사람을 끌어당기는 흐름
- 인맥/이성 접점 확장성
- 인연이 들어오는 방식

### Step 5. 화기와 살성
- 결핍
- 집착
- 확인 욕구
- 감정 리듬의 왜곡
- 과발현 시 문제

### Step 6. 유년/소한/중첩궁
- 올해 어떤 감정 주제가 활성화되는지
- 인연 유입 경로가 어디인지
- 관계 문제가 집중되는 시점이 있는지

### Step 7. 자녀궁
- 가까워질수록 드러나는 표현 매력
- 친밀감과 육체적 에너지
- 이성에게 비춰지는 후반부 매력

> 자녀궁을 앞 단계에서 제외한 이유는 중요하지 않아서가 아니라, data 자료상 부처궁·복덕궁·천이궁·관록궁보다 직접 해설 비중이 낮고, 역할상 ‘관계 구조의 핵심축’보다는 ‘매력 보조축’에 가깝기 때문이다.

## 4. 궁위별 핵심 해석 기준

| 궁위 | 해석 포인트 | 자기이해 번역 |
|---|---|---|
| 명궁 | 선천 기질, 성격의 중심 | 나는 처음에 어떤 사람처럼 보이는가 |
| 부처궁 | 연애관, 감정 태도, 배우자상 | 나는 관계에서 무엇을 원하고 어떻게 반응하는가 |
| 복덕궁 | 무의식, 정서 습관, 행복 기준 | 나는 혼자 있을 때 감정을 어떻게 처리하는가 |
| 천이궁 | 외부 환경, 대인관계, 만남의 장 | 나는 어디서 어떤 방식으로 사람을 만나기 쉬운가 |
| 관록궁 | 사회적 기준, 관계에 거는 조건 | 나는 상대에게 무엇을 기대하고 요구하는가 |
| 자녀궁 | 표현 매력, 친밀감, 성적 에너지 | 가까워질수록 어떤 매력이 드러나는가 |
| 노복궁 | 친구, 동료, 소개 연결 | 인연이 친구 네트워크를 통해 들어오는가 |
| 형제궁 | 친한 사람과 맺는 태도 | 가까운 관계에서의 기본 습관은 어떤가 |

---

## 5. 부처궁 해석 기준

부처궁은 이 가이드의 중심이다.

### 5-1. 부처궁에서 읽는 것
- 내가 감정을 받아들이는 방식
- 관계에서 편안함을 느끼는 구조
- 내가 반복적으로 반응하는 상대 유형
- 연애가 힘들어지는 핵심 이유

### 5-2. 부처궁이 약하다고 보는 기준
가이드 기준상:
- 살성 1~2개나 화기 1개로 나쁘다고 단정하지 않는다
- **살성 4개 이상**
- 그중 **짝성(화성-영성 / 경양-타라)**이 같이 작용
- 삼방사정에도 살성이 많이 모일 때  
→ 관계 유지 난도가 높다고 본다

### 5-3. 부처궁과 관록궁의 차이
- **부처궁** = 내가 진짜 감정적으로 원하는 것
- **관록궁** = 내가 상대에게 걸어놓는 조건과 기대

이 둘이 다르면:
- 끌리는 사람과 잘 맞는 사람이 다를 수 있다
- 관계 초반과 후반의 체감이 달라질 수 있다

---

## 6. 도화 해석 기준

### 6-1. 도화의 정의
도화는 단순히 “이성운”이 아니라  
**인맥, 접점, 유입, 인간관계 확장성**이다.

### 6-2. 도화로 읽는 것
- 사람을 끌어당기는 흐름이 있는가
- 이성과의 접점이 자연스럽게 생기는가
- 관계가 시작되기 쉬운 환경이 어디인가

### 6-3. 도화 해석 시 주의
- 도화 자체는 나쁘지 않다
- **도화 + 길성** → 관계 확장, 협력, 호감
- **도화 + 살성** → 도화살 작용, 나쁜 인연 반복 가능
- 최종 판단은 **복덕궁 강약**까지 함께 본다

---

## 7. 화기 해석 기준

화기는 “문제 별”이 아니라  
**결핍이 만들어내는 관계 집중 포인트**다.

### 7-1. 화기로 읽는 것
- 내가 계속 확인하고 싶은 감정
- 관계에서 쉽게 놓지 못하는 지점
- 사랑을 받는 방식보다, 갈구하는 방식

### 7-2. 궁위별 화기 번역

| 궁위 | 자미두수 의미 | 자기이해 번역 |
|---|---|---|
| 부처궁 화기 | 감정 의존 | 관계에서 상대 반응에 예민해짐 |
| 명궁 화기 | 존재 의미 결핍 | 관계가 자존감 문제로 연결되기 쉬움 |
| 복덕궁 화기 | 내면 평화 결핍 | 혼자 있을 때 불안과 공허가 커짐 |
| 자녀궁 화기 | 친밀감 집착 | 가까워진 뒤 감정이 더 깊게 걸림 |
| 질액궁 화기 | 편안함 의존 | 특정 사람 옆에서만 안정감을 느낌 |

### 7-3. 별별 화기의 해석 포인트

| 화기 | 핵심 갈증 | 관계 패턴 |
|---|---|---|
| 태양화기 | 인정받고 싶음 | 반응이 줄면 불안 |
| 거문화기 | 확신이 필요함 | 계속 진심을 확인하고 싶음 |
| 천기화기 | 소통 갈증 | 대화 끊기면 불안 |
| 천동화기 | 즐거움 갈증 | 무거워지면 답답함 |

---

## 8. 살성 해석 기준

살성은 “나쁜 성격”이 아니라  
**감정이 반응하는 리듬과 방식**을 보여준다.

| 살성 | 기본 리듬 | 자기이해 번역 |
|---|---|---|
| 화성 | 빠른 연소 | 빨리 뜨겁고 빨리 식는다 |
| 영성 | 은근한 불 | 천천히 보고 오래 계산한다 |
| 경양 | 빠른 결단 | 빨리 확신하고 끝까지 밀고 간다 |
| 타라 | 지연과 얽힘 | 끊지 못하고 오래 맴돈다 |

### 살성 해석 원칙
- 살성 하나만으로 문제라고 단정하지 않는다
- **어떤 감정 속도와 관계 반응을 만드는지**로 해석한다
- 화기와 결합하면 과발현 패턴을 읽는다

---

## 9. 매력 해석 기준

매력은 한 군데서만 읽지 않는다.  
반드시 **명궁 + 자녀궁 + 도화 + 주요 매력별**을 함께 본다.

### 9-1. 첫인상 매력
- 명궁
- 도화의 유무
- 천이궁의 외부 이미지

### 9-2. 가까워질수록 드러나는 매력
- 자녀궁
- 복덕궁
- 부처궁

### 9-3. 주요 매력별 해석

| 별 | 매력 해석 |
|---|---|
| 염정 | 정신적 카리스마, 강한 존재감 |
| 천량 | 신뢰감, 보호자적 안정감 |
| 탐랑 | 적극적 끌림, 감각적 매력 |
| 파군 | 낭만, 대담함, 새로움 |
| 문곡 | 재주, 정서적 여운 |
| 홍란 | 인연 활성화, 감정 접점 증가 |

---

## 10. “장점”이 아니라 “과발현”으로 읽는 기준

이 가이드에서 단점은 따로 떼어 읽지 않는다.  
같은 성향이 **건강하게 발현될 때**와 **과하게 발현될 때**를 나눠 본다.

| 건강한 발현 | 과발현 |
|---|---|
| 배려 | 눈치 봄, 자기감정 억압 |
| 솔직함 | 압박, 직설 과잉 |
| 안정감 | 무던함, 정체감 |
| 몰입 | 집착, 확인 욕구 |
| 신중함 | 지연, 결단 회피 |
| 추진력 | 성급함, 관계 소진 |

이 방식으로 해석해야  
“장점/단점”이 아니라  
**“왜 이 매력이 때로 문제로 바뀌는가”**를 설명할 수 있다.

---

## 11. 유년·소한 해석 기준

### 유년
- 올해 외부에서 들어오는 영향
- 관계 주제가 어떻게 활성화되는지
- 인연이 들어오는 시기보다, 감정이 강하게 흔들리는 흐름

### 소한
- 내가 올해 무엇을 원하고 추구하는지
- 내 감정의 방향성

### 중첩궁
- 소한 명궁 아래에 깔린 선천 명반 궁위
- 인연 유입 경로를 해석할 때 사용

| 중첩궁 | 해석 |
|---|---|
| 관록궁 | 직장/업무 환경 |
| 노복궁 | 친구 소개 |
| 천이궁 | 외부 활동/이동 |
| 부모궁·형제궁 | 연장자/지인 연결 |

---

## 12. 최종 출력으로 번역하는 법

명반을 바로 설명하지 말고 아래 4단으로 번역한다.

### 1단: 성향
- 나는 관계에서 어떤 사람인가

### 2단: 매력
- 나는 어떤 방식으로 사람을 끌어당기는가

### 3단: 과발현
- 그 매력이 과해질 때 무슨 문제가 생기는가

### 4단: 관계 환경
- 나는 어떤 흐름과 환경에서 자연스럽게 빛나는가

---

## 13. 이 가이드로 만들 수 있는 콘텐츠 단위

| 콘텐츠 단위 | 해석 근거 |
|---|---|
| 나의 관계 기본값 | 명궁 + 복덕궁 + 부처궁 |
| 내가 끌리는 관계 | 부처궁 + 관록궁 |
| 나의 매력 구조 | 명궁 + 자녀궁 + 도화 |
| 과몰입/확인 욕구 패턴 | 화기 + 부처궁 + 복덕궁 |
| 인연 유입 방식 | 천이궁 + 노복궁 + 중첩궁 |
| 올해 관계 흐름 | 유년 + 소한 + 부처궁 |
