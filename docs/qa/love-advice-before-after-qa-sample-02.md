## 5. 결론

After가 Before보다 안전하고 제품 문체에 더 가깝다.

```text
Before: 별/궁 이름이 남고, 불안·기대 패턴 설명이 내부 용어에 기대는 경향이 있음.
After: 내부 용어가 제거되고, 신뢰 형성·조건 정리·감정 반응 조절이 사용자 문장으로 바뀜.
```

현재 기준에서는 rule 수를 조정하지 않는다.

```text
권장값 유지:
- 섹션 1~5: 최대 2개
- 섹션 6: 최대 3개
- 전체: 최대 6개
```
