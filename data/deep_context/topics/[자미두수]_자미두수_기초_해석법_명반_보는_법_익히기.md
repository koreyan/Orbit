# [자미두수] 자미두수 기초 해석법 / 명반 보는 법 익히기

안녕하세요 월천입니다.오늘은 자미두수의 구조나 기본적 풀이에 대하여 가장 단순하게 설명을 드리는 시간을 가져보려고 합니다.자미두수 기초나 명반 해석법, 명반 보는 법은 명반을 반복적으로 보면서 익숙해지는 것이니 너무 어렵게 생각할 필요는 없고 단지 시간이 필요할 뿐입니다.아직 자미두수가 어떻게 생긴 것인가 모르는 분들이 많으셔서 올려보네요.이렇게 전체적인 면을 먼저 보여드리고 자세한 내용은 계속 연재해볼 계획이니 지금은 이 글을 보시며 "그냥 이런게 있구나~" 하고 보아주시면 되겠습니다.1) 명반자미두수를 보려면 제일 먼저 필요한 것이 바로 명반입니다.명반은 "나의 운명이 담긴 그릇" "운명을 도표한 도표" 라고 보시면 좋겠습니다.여기서 중요한 것은 태어난 시간을 정확히 알아야 명반을 그릴 수 있다. 라는 부분입니다.명리의 경우는 생일 까지만이라도 표현이 되지만 자미두수는 태어난 시간까지 정확해야 명반을 작성할 수 있습니다. 그래서 시간의 경계에 태어난 분들의 경우는 양쪽 명반의 특징을 대조하여 태어난 시간을 찾기가 쉬운 부분도 있습니다.
2) 자미두수 해석 방법자미두수를 이해하는데는 크게 2가지 개념을 알아야 합니다.인생의 12가지 요소를 말하는 12궁 / 명반에 배포되는 각종 별들 이 그것입니다.12궁은 12가지 영역을 나타내고 이 궁에 들어가는 별들의 삼합이 되는 자리. 충이 되는 자리의 별들을 조합하여 해당 궁의 의미를 파악하게 됩니다.궁의 삼합이 되는 자리 + 대충궁을 전문용어로 삼방사정 이라고 부릅니다.
2) 명반의 12궁자미두수의 명반을 작성하면 이제 읽어내는 법을 익혀야 합니다.자미두수는 크게 "궁 (자리의 의미)" 와 "성 (별들)" 을 조합하여 운명을 읽게 됩니다.예를 들면 명궁(자기자신)의 자리에 자미성이 있으면 성격이 왕과 같다. 즉, 자존감이 강하고 무언가 지휘하고 경영하는 것. 리더십이 있는 사람이다. 라고 해석을 합니다.이처럼 인생의 12가지 삶의 요소를 나타내는 자리를 12궁 이라고 부릅니다.12궁명궁 : 자기자신형제궁 : 형제와 가까운 친구, 동업자부처궁 : 연애운, 배우자자녀궁 : 자녀, 나의 창작물재백궁 : 재물의 안정성질액궁 : 건강천이궁 : 사회활동, 유학, 여행노복궁 : 아랫사람, 애완동물관록궁 : 직업환경의 안정성전택궁 : 주택과 부동산, 유산, 목돈복덕궁 : 정신세계, 정신력, 타고난 복부모궁 : 부모님, 윗사람, 선생님

3) 자미두수의 별들자미두수는 14개의 주성이 큰 틀을 만들고 6개의 길성 (편한별) 6개의 흉성 (힘든별)이 모여서 길흉을 만들어내게 됩니다. 아래 가장 대표적인 의미를 적어봅니다.14주성자미성：황제의 기품이 있다. / 제왕천기성：총명하고 재주가 많다. / 지혜, 변화태양성：친절하고 시원시원함 / 명예, 나눔무곡성：강하고 결단력이 있다. / 꼼꼼함, 결단력천동성：온화하고 편안함을 추구한다. / 부드러움, 아동, 긍정성염정성：주도권. 결정권을 가지고 싶어하고 깔끔하다./ 주도권, 완벽주의천부성：웃는 얼굴이 돈을 가져다 준다. / 포용력, 생산 , 관리태음성：낭만적이고 감정이 풍부하다./ 낭만, 완벽주의, 꿈과 희망, 변화, 변덕탐랑성：팔방미인으로 재주가 많다./ 욕망, 다재다능, 호기심거문성：혼란속에서 규칙을 세운다. 비교를 잘한다 / 구설수, 몰입, 의심. 분석력천상성：배려가 깊다. 성실하다. / 배려, 성실, 화합, 대리인, 책임자천량성：노련하고 여유로움 / 노하우, 노인, 지원, 무게감칠살성：용감하고 적극적이다. / 전투, 결단, 추진, 적극적파군성：창의적이고 모험적이다. / 소모, 창조, 모험
6길성 (명반에서 파란색 별들)천괴 : 국가, 조직에 의한 행운, 선택, 좋은 인연, 직접적으로 느껴지는 행운천월 : 환경적으로 좋아지는 행운, 간접적으로 좋은 영향이 생겨난다.좌보 : 도움을 주는 사람, 동료의 도움, 환경의 도움이 생기는 힘우필 : 비슷한 나이대의 사람의 지지, 힘을 실어준다. 환경이 유리해진다.문창 : 표현력, 창조력, 자기어필을 잘한다. 센스가 있다.문곡 : 글을 잘쓴다. 공부를 잘한다. 시험에 유리하다.
6살성 (명반에서 빨간색 별들)경양 : 외부적인 어려움, 단절타라 : 외부적인 어려움, 고난화성 : 내부적인 어려움, 불만 표현영성 : 내부적인 어려움, 불안지공 : 정신적인 공허함. 새로운 아이디어, 특이한 취향지겁 : 물질적인 소모. 함정에 빠지다. 잘못된 선택. 자기희생
그 외의 참고할 별들녹존 : 실질적인 재물. 고정관념. 방어적 본능천마 : 활동성, 움직임, 변화천형 : 자기반성, 형벌, 권력천요 : 부드러움, 적응력, 우유부단잡성이 외의 모든 별들은 잡성에 들어가고 잡성은 참고만 할 뿐 그 자체로 큰 역할이 없다.주성과 비슷한 성질의 잡성이 그 역할을 강화 시키거나 잡성과 길성 흉성이 비슷한 의미들이 합쳐지면 그때의 나름의 의미가 생깁니다.
4) 명궁에 14주성이 없다면? 차성안궁을 한다.

주성이 하나의 궁에 들어가는 경우는 총 3가지로1) 1개만 들어가는 경우2) 2개 들어가는 경우 (별의 의미가 강화되거나 대립, 보완)3) 하나도 들어가지 않는 경우하나도 들어가지 않는 경우는 대충궁에 있는 자리의 별이 해당 궁의 주인 역할을 한다고 풀이를 합니다.아래는 차성안궁이 이루어지는 궁들을 정리한 내용입니다.(예제 : 명궁에 14주성이 없으면 천이궁의 14주성이 명궁의 별 역할을 한다)명궁 - 천이궁형제궁 - 노복궁부처궁 - 관록궁자녀궁 - 전택궁재백궁 - 복덕궁질액궁 - 부모궁5) 명반해석에 대하여명반프로그램이나 사이트에서 해석하고자 하는 궁의 자리를 클릭하면 그 궁으로 화살표 모양이 생기게 됩니다.여기서 그 궁의 주성과 삼방 (삼합을 하는 자리 - 빨간색 삼각형 화살표)사정궁 (지지를 충을 하는 자리 - 화살표 반대편 자리)을 보고 주성과 길성. 살성의 역할과 상호작용을 해석합니다.별들의 조합으로 특수한 격국이 형성이 되는 경우도 있지만 대개의 경우는 별의 성질로 해석을 하며 간단하게 아래와 같이 단식판단을 할 수 있습니다.길성 + 흉성이 섞여 있다 = 좋은 점도 있고 나쁜점도 있다길성이 4개 이상 = 좋은 의미가 크다흉성이 4개 이상 = 조심할 부분이 있다그 외에 별의 사화 / 짝성의 역할 / 별들의 힘의 세기 / 특정격국등을 참고하여 해석을 하게 됩니다.자미두수의 구조와 해석 흐름에 대해서 최대한 간결하게 설명을 한 것이라 이 중간에 빠진 내용도 많지만 자미두수가 어떤 것인지 알리고자 하는 마음에 작성한 것이니 자세한 것은 다음 글에서 채워가도록 하겠습니다.긴글 읽어주심에 감사드립니다 ^^
